# 2025 国赛 B 题完整实验结果

## 1. 完成范围

本仓库已经完成以下四个工作包：

1. 问题 1：双光束干涉模型推导与代码映射；
2. 问题 2：附件 1、2 的 SiC 厚度计算与可靠性分析；
3. 问题 3：附件 3、4 的多光束判定及 Si 厚度计算；
4. 问题 3 补充：SiC 多光束影响复核、基频修正和修正后结果。

## 2. 最终数值结果

推荐在论文正文中报告：

- **SiC 外延层厚度：** $d_{SiC}=7.476\,\mu m$，推荐不确定区间 [7.4388, 7.5145] μm；

“推荐区间”取统计 Bootstrap 区间与核心敏感性区间的并集；扩展压力测试区间分别为：

- SiC：[7.4017, 7.5527] μm；
- Si：[3.4288, 3.5665] μm。

## 3. 多光束干涉结论

### 3.1 SiC 附件 1、2

分类：`non_sinusoidal_but_not_material`。

- 峰—谷厚度中位相对差：0.076%；
- 二次谐波/基频中位比：0.0178；
- 条纹可见度中位数：0.0885；
- 基频修正量：-0.0049 μm。

结论：高阶谐波模型能改善拟合，但峰谷厚度一致、二次谐波和条纹可见度较低；现有证据不足以认定多光束对厚度造成实质影响。

### 3.2 Si 附件 3、4

分类：`significant_multi_beam`。

- 峰—谷厚度中位相对差：1.840%；
- 二次谐波/基频中位比：0.0671；
- 条纹可见度中位数：0.6523；
- 相对于全部方法中位数的修正量：-0.0261 μm。

结论：多项证据同时超过阈值，多光束干涉会使波谷/全波形双光束估计产生系统偏差；最终厚度采用基频隔离后的峰坐标估计。

## 4. 可靠性结果

### 分块 Bootstrap

光谱相邻采样点相关，因此按连续点块重采样，而不是逐点独立重采样。两组实验均完成 400 次有效重复：

- SiC 标准误：0.0081 μm；
- Si 标准误：0.0179 μm。

### 核心敏感性

核心情景包括折射率 ±0.5%、入射角 ±0.2°、平滑窗口、基线阶数、包络窗口及峰值阈值：

- SiC 核心范围：[7.4388, 7.5145] μm；
- Si 核心范围：[3.4547, 3.4961] μm。

### TMM 参数扫描

衬底掺杂、复折射率和吸收参数未由附件直接给出，因此 TMM 扫描只用于说明参数可辨识性：

- SiC TMM 厚度扫描范围：[7.3714, 7.6540] μm；
- Si TMM 厚度扫描范围：[3.5437, 3.7651] μm。

该范围明显依赖衬底光学参数，故不把单个 TMM 最优值直接作为最终厚度。

## 6. 结论边界

最终厚度依赖所采用的外延层折射率模型。Bootstrap 反映测量噪声与光谱局部相关性，不包含所有光学常数系统误差；推荐区间已额外并入核心参数敏感性。TMM 中衬底参数未被唯一识别，因此只能用于验证多光束模型的合理性和结果范围。
