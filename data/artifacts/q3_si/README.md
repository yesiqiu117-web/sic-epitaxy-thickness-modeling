# Intermediate artifacts: q3_si

These files are automatically generated by the existing numerical pipeline.
They are audit products only: exporting them does not change any model, parameter, or final thickness.

## Dataset-level evidence

### `si_10deg/`

- `preprocessing.csv`: raw input within the modeling band, Hampel-cleaned, smoothed, baseline, residual, envelope, normalized fringe, and outlier flags.
- `fft_spectrum.csv`: complete FFT spectrum, candidate peaks, and selected fundamental.
- `peak_regression.csv` / `valley_regression.csv`: extrema coordinates, assigned interference orders, fitted orders, and residuals.
- `hilbert_phase.csv`: filtered fundamental signal, unwrapped phase, fitted phase, and residuals.
- matching `*_summary.json`: compact numerical summaries.

### `si_15deg/`

- `preprocessing.csv`: raw input within the modeling band, Hampel-cleaned, smoothed, baseline, residual, envelope, normalized fringe, and outlier flags.
- `fft_spectrum.csv`: complete FFT spectrum, candidate peaks, and selected fundamental.
- `peak_regression.csv` / `valley_regression.csv`: extrema coordinates, assigned interference orders, fitted orders, and residuals.
- `hilbert_phase.csv`: filtered fundamental signal, unwrapped phase, fitted phase, and residuals.
- matching `*_summary.json`: compact numerical summaries.

## Joint model evidence

`joint_models/` contains observed, fitted, and residual values for the two-beam, three-harmonic, and TMM models, together with one JSON summary per model.

Final publication-ready figures remain in `outputs/figures/`; final summary tables remain in `outputs/tables/`.
