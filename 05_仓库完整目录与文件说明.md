# 碳化硅/硅外延层厚度反演仓库：完整目录与文件说明

## 1. 总体目录

```text
SiC_epitaxy_modeling_repo/
├── .github/
├── configs/
├── data/
├── docs/
├── outputs/
├── scripts/
├── src/epitaxy/
├── tests/
├── .gitignore
├── Makefile
├── README.md
├── pyproject.toml
└── requirements.txt
```

文件类型：

- **核心源码**：实现模型与算法；
- **配置文件**：控制参数，不改源码即可调整实验；
- **输入数据**：官方附件；
- **自动输出**：运行后生成，可删除重算；
- **说明文档**：人工维护；
- **测试文件**：验证核心函数；
- **占位文件**：让 Git 保留空目录；
- **缓存文件**：不应提交。

## 6. `scripts/`

| 文件 | 作用 |
|---|---|
| `run_all.py` | 默认读取已有 JSON 重建总表；`--recompute` 从四附件完整重算 |
| `run_q2_sic.py` | 单独运行 SiC，即问题二和问题四 |
| `run_q3_si.py` | 单独运行 Si，即问题三 |
| `generate_synthetic.py` | 生成已知厚度的合成数据 |
| `smoke_test.py` | 合成数据端到端冒烟测试 |

常用命令：

```bash
python scripts/run_all.py --quiet
python scripts/run_all.py --recompute --quiet
python scripts/run_q2_sic.py --config configs/q2_sic.yaml
python scripts/run_q3_si.py --config configs/q3_si.yaml
PYTHONPATH=src pytest -q
```

## 20. 修改后的运行建议

| 修改内容 | 至少运行 |
|---|---|
| 只改 Markdown | 无需重算 |
| 改 YAML | 完整重算 |
| 改预处理或估计 | `pytest -q` 后完整重算 |
| 改折射率、Fresnel、TMM | 测试、冒烟测试、完整重算 |
| 改最终估计器 | 完整重算并核对 summary JSON 和总表 |
| 只重建总报告 | `python scripts/run_all.py --quiet` |
