# 实验方案

## 1. 总体原则

实验不采用“只找两个峰直接代公式”的单线方案，而采用四层证据链：

1. **数量级证据**：FFT、自相关、相邻峰间距；
2. **几何证据**：修正波数—峰级次稳健回归；
3. **全谱证据**：双角度联合拟合；
4. **机理证据**：Airy/TMM、多光束谐波和残差结构。

每个最终厚度必须同时报告：点估计、双角度差异、方法间差异、分频段差异和置信区间。

## 3. 问题 2 的实验流程

### 3.1 数据诊断

对附件 1、2 分别输出：

- 波数是否单调、是否等间距；
- 缺失值、重复波数；
- 首点是否为异常零值；
- 反射率最小值、最大值、超过 100% 的比例；
- Hampel 异常点数量。

超过 100% 的反射率只记录，不直接裁剪。后续模型用慢变背景、增益和偏置吸收标定误差。

### 3.2 预处理对照

主结果默认使用 SG 窗口 21。对 11、21、31、41 重复 E2–E5，记录厚度变化：

\[
S_{\mathrm{SG}}=\frac{\max d_w-\min d_w}{\operatorname{median}d_w}.
\]

若窗口改变导致峰数突变或厚度明显漂移，应缩小平滑强度并检查基线。

### 3.3 初值与候选排歧

FFT 返回前若干候选频率，不只返回最高峰。候选厚度应满足：

- 在两个角度均出现邻近候选；
- 相邻峰法与 FFT 数量级一致；
- 候选的 2 倍、3 倍频不能被误当作基频。

### 3.4 单角度回归

峰和谷分开回归：

\[
m_j=2d x_j+b+\varepsilon_j.
\]

若相邻峰距约为中位峰距的 2 倍，则将级次增量设为 2，以处理漏峰。最终报告：

- `d_peak`、`d_valley`；
- 峰谷相对差；
- 10°、15° 相对差；
- 回归残差与可疑漏峰位置。

### 3.5 双角度联合拟合

两个角度共享厚度 `d`，每个角度独立设置相位、基线和振幅包络。使用多相位初值，防止局部极小。

最终结果优先级：

1. 联合拟合厚度；
2. Hilbert 相位法与峰级次回归必须给出相近结果；
3. 若差异超过 Bootstrap 区间，不能简单平均，应回查折射率、频段和多光束影响。

## 4. 问题 3 的多光束判定

### 4.1 判定证据

对附件 3、4 分别建立双光束与 TMM 模型，采用以下证据：

1. `H2/H1`、`H3/H1` 明显高于碳化硅或噪声背景；
2. 双光束残差含稳定的二次/三次谐波；
3. TMM 相对双光束 `ΔBIC = BIC_two - BIC_tmm > 10`；
4. TMM 在连续波数块交叉验证中仍有更低误差；
5. TMM 使 10°、15° 的厚度结果更一致。

其中 `ΔBIC > 10` 只作为强证据之一，不作为唯一必要条件。

### 4.2 模型对照

- M1：双光束余弦模型；
- M2：无吸收 Airy；
- M3：含吸收 TMM；
- M4：TMM + 标定增益/偏置/线性背景。

应控制复杂度，禁止无约束同时拟合大量折射率和 Drude 参数。

## 5. 不确定度实验

### 5.1 分块 Bootstrap

光谱点相邻相关，采用连续波数块或完整条纹块重采样，不采用逐点独立 Bootstrap。

默认块长 80 个采样点，重复 200 次。输出中位数和百分位置信区间。

### 5.2 参数敏感性

建议至少计算：

- 折射率整体缩放：`n × (1±0.2%)`、`n × (1±0.5%)`；
- 入射角：`θ±0.1°`、`θ±0.2°`；
- 频段边界：上下边界各移动 50–100 cm^-1；
- SG 窗口：11、21、31、41；
- 峰显著度：默认值上下浮动 20%。

最终用 Tornado 表或排序表给出主要误差源。

---

## 6. 结果验收规则

以下规则用于内部自检，不应机械写成“绝对标准”：

- 双角度差异越小越好；若超过约 3%，应回查模型或数据；
- 峰法、Hilbert、联合拟合应在各自不确定度范围内相容；
- TMM 修正量若小于总不确定度，可认为双光束近似足够；
- TMM 若显著降低 BIC、残差谐波和角度差异，可认为多光束模型必要；
- 最终厚度不能只报告单个数字，必须附置信区间和稳定性证据。
