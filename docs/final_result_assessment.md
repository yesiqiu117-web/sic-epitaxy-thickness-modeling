# 最终结果独立评估说明

> 本说明基于当前仓库中的源代码、四个原始附件、结果表、拟合图和自动测试进行复核。它评估的是“当前代码与数据能够支持到什么程度”，不是对样品真实厚度的外部标定证明。

## 1. 总体评价

当前仓库已经形成一条完整且可复现的实验链：

```text
原始 Excel
→ 数据清洗与基线/包络处理
→ 修正波数
→ FFT 初值
→ 峰/谷级次回归
→ Hilbert 相位验证
→ 双角度联合模型
→ 谐波与多光束诊断
→ Bootstrap 与敏感性分析
→ 最终结果汇总
```

工程实现较完整：项目结构清晰，配置、源代码、结果与测试相互分离；当前自动测试为 **12 项全部通过**。

但从数学建模结论的严谨性看，两个材料的可信度并不相同：

- **SiC 的最终厚度结果较稳健**，适合作为论文主结果；
- **Si 的最终厚度结果可作为“基频峰位稳健估计”**，但不宜表述为已经通过物理 TMM 唯一反演并彻底消除了多光束影响。

## 2. SiC 最终结果评估

仓库当前结果为：

\[
\hat d_{\mathrm{SiC}}=7.4763\ \mu\mathrm m.
\]

分块 Bootstrap 区间为：

\[
[7.4580,\ 7.4919]\ \mu\mathrm m,
\]

综合核心敏感性后的推荐区间为：

\[
[7.4388,\ 7.5145]\ \mu\mathrm m.
\]

### 2.1 支持该结果的证据

1. 10° 与 15° 两个角度的最终估计相对差为 **0.434%**；
2. 峰、谷和 Hilbert 三类基频方法在完整主频段内集中在约 7.45–7.49 μm；
3. 峰—谷厚度中位相对差仅为 **0.076%**；
4. 二次谐波与基频的中位幅值比为 **0.0178**，较低；
5. 折射率 ±0.5%、角度 ±0.2°、平滑窗口、基线阶数和包络窗口扰动后，核心结果仍位于 7.4388–7.5145 μm。

### 2.2 仍需保留的结论边界

1. 双光束全谱拟合、三谐波拟合和基频共识结果并不完全相同：约为 7.419、7.541 和 7.476 μm；
2. 某些较窄验证频段出现明显漂移，说明短频段中的周期数不足、色散误差或谐波混叠仍会影响结果；
3. Bootstrap 固定了基频重构和折射率模型，因此主要反映条件于当前模型的随机波动，不包含全部光学常数系统误差；
4. 结果保留到 4 位小数会显得过度精确。

### 2.3 推荐论文写法

正文推荐写为：

> 采用双角度基频峰谷回归与 Hilbert 相位交叉验证，得到 SiC 外延层厚度约为 **7.48 μm**。统计 Bootstrap 区间为 7.458–7.492 μm；将折射率和预处理敏感性并入后，推荐不确定区间为 **7.44–7.51 μm**。

不建议只写“7.4763 μm”而不说明模型依赖和不确定区间。

## 3. Si 最终结果评估

仓库当前采用峰坐标估计：

\[
\hat d_{\mathrm{Si}}=3.4720\ \mu\mathrm m.
\]

分块 Bootstrap 区间为：

\[
[3.4502,\ 3.5198]\ \mu\mathrm m.
\]

扩展压力测试范围为：

\[
[3.4288,\ 3.5665]\ \mu\mathrm m.
\]

### 3.1 支持该结果的证据

1. 两个角度的峰位结果分别为 3.4767 μm 和 3.4674 μm，相对差仅为 **0.268%**；
2. Si 的条纹可见度较高，峰谷厚度差达到 **1.840%**，明显高于 SiC；
3. 二次谐波与基频的中位幅值比为 **0.0671**；
4. 三谐波模型相对双光束模型的 BIC 有显著改善，说明条纹并非理想正弦；
5. 采用峰位而不是波谷或纯正弦全谱拟合，可以降低尖锐 Airy 波形对极小值定位的影响。

### 3.2 当前最重要的风险

不同估计路线仍存在较明显差异：

- 峰位估计：约 3.47 μm；
- 谷值/Hilbert：约 3.52–3.55 μm；
- 双光束全谱拟合：约 3.63 μm；
- 三谐波全谱拟合：约 3.64 μm；
- 当前有效参数 TMM：约 3.72 μm。

因此，3.472 μm 是一个**方法明确、双角度一致的稳健估计**，但当前证据尚不足以证明它是由完整物理 TMM 唯一确定的真实值。

此外，当前 TMM 使用的是有效衬底复折射率参数。题目没有给出衬底掺杂浓度、载流子有效质量、碰撞频率和完整吸收谱，所以 TMM 参数不可唯一辨识。

### 3.3 推荐论文写法

建议写为：

> Si 光谱呈现显著非正弦和高可见度条纹，峰谷估计出现系统分离。为减弱多次反射导致的波谷偏移，本文在基频隔离后采用双角度峰坐标回归，得到外延层厚度约为 **3.47 μm**，条件 Bootstrap 区间为 3.450–3.520 μm。考虑频段与折射率压力测试后，结果可保守表述为 **3.43–3.57 μm**。

建议使用“多光束稳健估计”或“基频峰位修正”，不要直接写成“已通过 TMM 完全消除多光束影响”。

## 4. 多光束判定的评估

当前判定综合使用：

- 峰谷厚度差；
- 二次谐波幅值比；
- 条纹可见度；
- 双光束与三谐波模型 BIC 差。

这比只依据一项指标更可靠。当前结论“Si 存在显著多光束影响，而 SiC 的影响不构成主要厚度偏差”具有合理性。

不过需要说明：

1. 阈值 1%、0.04、0.12 和 ΔBIC=10 是工程判据，尚未通过大量仿真进行统计标定；
2. BIC 默认残差近似独立，而当前各模型残差的一阶自相关接近 1，说明 BIC 的绝对证据强度可能被放大；
3. 三谐波模型证明的是“波形明显非正弦”，并不能单独证明所有非正弦成分都来自多次内部反射；
4. 仍可能混入基线模型误差、仪器响应、吸收变化和复 Fresnel 相位变化。

## 5. Bootstrap 与不确定度评估

当前采用连续残差块 Bootstrap，比逐点独立重采样更合理。但其区间应解释为：

> 在当前折射率、频段、预处理和基频选择固定条件下的统计不确定度。

它尚未完整覆盖：

- 折射率公式选择误差；
- 基频识别或谐波混叠；
- 多光束模型选择误差；
- 衬底复折射率不可辨识性；
- 块长度选择误差。

因此最终论文应优先报告“Bootstrap 区间 + 敏感性推荐区间”，不能把 Bootstrap 区间称为覆盖全部误差来源的最终置信区间。
