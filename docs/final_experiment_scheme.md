# 最终实验方案与消融设计

## 一、实验目标

实验不仅要给出厚度数值，还要回答三类问题：

1. 厚度主值是多少；
2. 多光束干涉是否会使双光束结果产生实质偏差；
3. 结果对噪声、折射率、入射角、预处理和拟合频段有多敏感。

## 二、问题 2：SiC 实验

### 主实验

1. 读取附件 1、2；
2. 删除异常首点，执行 Hampel、Savitzky–Golay、稳健基线和包络归一化；
3. 在 `1500–4000 cm⁻¹` 透明区计算修正波数；
4. 分别运行 FFT、峰级次回归、谷级次回归和 Hilbert 相位法；
5. 以峰、谷和 Hilbert 的跨角度中位数作为基频共识结果；
6. 用双角度双光束、三谐波和 TMM 模型进行拟合比较；
7. 运行 400 次连续块 Bootstrap；
8. 运行核心敏感性和扩展压力测试。

### 多光束复核

同时检查：

- 峰和谷给出的厚度是否分离；
- `H2/H1` 是否显著；
- 条纹可见度是否足够高；
- 三谐波模型是否显著改善 BIC。

只有前三项物理证据同时达到阈值，才认为多光束对厚度造成实质影响，避免把基线误差或折射率模型误差误判成多光束。

## 三、问题 3：Si 实验

### 多光束判定

Si 数据低波数区条纹可见度高、波形明显非正弦，因此采用多证据联合判定。若判定为显著多光束：

1. 不直接用峰谷混合或纯余弦全谱拟合作为最终厚度；
2. 先在修正波数域隔离基频；
3. 只使用峰坐标回归，因为高阶内部反射主要改变波谷和波形尖锐程度，峰的周期位置更稳定；
4. 两个角度共享材料折射率模型，并以峰法结果的中位数作为最终值。

### 物理模型验证

并行运行含吸收的 TMM 模型和衬底有效光学参数扫描。由于附件未给出衬底掺杂和复折射率，TMM 最优值只作为可辨识性与敏感性证据，不直接替代峰坐标主结果。

## 四、不确定度实验

### 1. 连续块 Bootstrap

- 重复次数：400；
- 块长：80 个采样点；
- 重采样对象：基频重构之外的连续残差块；
- 输出：标准误和百分位 95% 区间。

连续块方法保留光谱相邻点的相关性，优于逐点独立重采样。

### 2. 核心敏感性

核心情景：

- 折射率实部 ±0.5%；
- 入射角 ±0.2°；
- SG 窗口 11、31、41；
- 基线阶数 2、4；
- 包络窗口 201、401；
- 峰值 prominence 0.18、0.32。

### 3. 扩展压力测试

- 折射率实部 ±1%；
- 多组拟合波段；
- TMM 衬底有效折射率和吸收参数网格。

## 五、最终结果选择原则

| 数据 | 最终估计器 | 原因 |
|---|---|---|
| SiC 附件 1、2 | `fundamental_consensus` | 峰谷一致，多光束无实质厚度影响 |
| Si 附件 3、4 | `peak_only` | 多光束显著，峰坐标比波谷和全波形更稳定 |

推荐区间定义为：

```text
推荐区间 = Bootstrap 95% 区间 ∪ 核心敏感性区间
```

扩展压力测试区间单独报告，不与统计置信区间混写。
