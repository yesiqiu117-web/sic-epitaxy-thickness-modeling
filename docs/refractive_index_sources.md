# 折射率与介电函数模型说明

## SiC 外延层

配置 `configs/q2_sic.yaml` 使用 4H-SiC 的一声子 TO–LO 因子化介电函数：

- 高频介电常数：`epsilon_inf = 6.52`；
- 横向光学声子：`TO = 797.7 cm^-1`；
- 纵向光学声子：`LO = 992.1 cm^-1`；
- 阻尼参数在配置中可调。

1500–4000 cm⁻¹ 位于主要声子带上方，适合作为问题 2 的主拟合透明区。800–1100 cm⁻¹ 附近不能使用简单常数实折射率，应使用复介电函数并考虑吸收。

## Si 外延层

配置 `configs/q3_si.yaml` 使用 Chandler–Horowitz 与 Amirtharaj 的室温中红外硅折射率公式：

\[
n^2(\lambda)=11.67316+\frac{1}{\lambda^2}
+\frac{0.004482633}{\lambda^2-1.108205^2},
\]

其中波长 \(\lambda\) 的单位为 μm。代码在 `src/epitaxy/optics/refractive_index.py` 中实现。

## 衬底参数边界

题目未给出四块样品衬底的掺杂浓度、载流子有效质量、碰撞频率或复折射率表。因此：

- 外延层折射率使用文献公式；
- 衬底仅使用可配置的有效 `n_offset`、`n_scale` 与 `kappa`；
- TMM 输出标记为敏感性分析，不将有效参数解释为唯一真实材料参数；
- 论文最终实验应扫描一组合理的 Drude/吸收参数区间，并报告厚度稳定性。
